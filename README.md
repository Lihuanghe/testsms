# testsms
短信网关测试客户端-命令行工具

# 命令行工具
```

usage: Options
 -attime <arg>   At_Time
 -c <arg>        config File
 -dcs <arg>      msg-fmt
 -h              help info
 -msgsrc <arg>   msgsrc
 -sid <arg>      server id
 -spcode <arg>   spcode
 -tel <arg>      telephone
 -txt <arg>      SMS Content
 -wait <arg>     wait time to exit
```
